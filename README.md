## #1.Setup
Windows :
```text
do downloaded file
```
Mac :
```shell
$ cd path of downloaded file of folder
$ chmod 700 downloaded file name
$ ./downloaded file name
```
Linux :
```shell
$ cd path of downloaded file of folder
$ chmod 700 downloaded file name
$ ./downloaded file name
```
